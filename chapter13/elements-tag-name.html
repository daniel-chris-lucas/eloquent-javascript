Elements by tag name

The getElementsByTagName method returns all child elements with a given tag name. Implement
your own version of it as a regular nonmethod function that takes a node and a string
(the tag name) as arguments and returns an array containing all descendant element nodes
with the given tag name.

To find the tag name of an element, use its tagName property. But note that this will return
the tag name in all uppercase. Use the toLowerCase or toUpperCase string method to compensate
for this.

This exercise is logged into the developer tools console using console.log()
